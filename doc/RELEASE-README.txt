This is a release of https://github.com/ahamlinman/xt, a cross-format
translator for serialized data formats.

xt is the binary release of xt itself. It should require no additional setup
to execute, as long as you've obtained the correct release for your platform.
Try "./xt --help" for a usage summary.

xt.1 is the xt manual page. Try "man ./xt.1" to read it.

LICENSES.html provides copyright licenses and source code links for xt and the
third-party software it incorporates.
